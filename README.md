# 보소사진관 관리자

사진관 고객, 방문, 예약, 결제, 보정 사진 기록을 관리하는 로컬 웹앱입니다.

## 실행 방법

1. 이 폴더에서 로컬 서버를 실행합니다.
2. 브라우저에서 `http://127.0.0.1:5173/index.html`로 접속합니다.

PowerShell 예시:

```powershell
python -m http.server 5173 --bind 127.0.0.1
```

## 주요 기능

- 고객 고유번호 자동 발급
- 이름, 전화번호, 아이 이름, 고객번호 검색
- 고객별 방문 회차와 촬영 기록 관리
- 촬영일, 촬영종류, 촬영상품/패키지 기록
- 예약금, 잔금, 결제수단, 결제 직원 기록
- 예약 일정 관리
- 월별 매출과 촬영종류별 매출 확인
- 보정 사진 업로드와 방문 기록 연결
- JSON 백업/복원, CSV 내보내기
- Google Sheets 양방향 연동
- Google Calendar 예약 동기화

## Google Sheets 연동

`google-apps-script.gs` 파일 내용을 Google Sheets의 Apps Script에 붙여 넣고 웹 앱으로 배포합니다.
배포 URL을 프로그램의 `연동/백업` 화면에 저장하면 다음 기능을 사용할 수 있습니다.

- 시트로 보내기
- 시트에서 가져오기
- 예약을 캘린더로 보내기

자세한 방법은 `구글시트_연동방법.txt`를 참고하세요.

## 주의

사진 파일은 브라우저 저장소에 보관됩니다.
Google Sheets에는 사진 파일 자체가 아니라 사진 개수만 저장됩니다.
